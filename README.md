# WTP · Калькулятор баланса карты

Внутренний инструмент саппорта WantToPay. Саппорт вставляет таблицу транзакций из админки (Nova) — инструмент считает итоговый расчётный баланс карты с учётом пополнений, трат, комиссий, выводов и деклайнов.

## Как пользоваться

1. Открой страницу: **https://novikoffalex.github.io/wtp-balance-calc/**
2. В админке выдели таблицу «Транзакции» или «Транзакции API» по карте, скопируй (Ctrl/Cmd+C).
3. Вставь в поле — баланс посчитается автоматически.

## Логика расчёта

`Баланс = Пополнения − Траты(auth) − Комиссии − Выводы`

| Тип | Влияние |
|-----|---------|
| `TOP_UP` / Пополнение / Deposit | **+** |
| `AUTH` / трата / Debit | **−** |
| `FEE` / Комиссия | **−** |
| `WITHDRAW` / вывод | **−** |
| `DECLINE` / Отклонено | **0** (учитывается только их комиссия) |
| `CREATION` / Выпуск карты | показывается отдельно, нейтрально |

Распознаёт оба формата таблиц админки (EN/RU метки), запятую и точку как десятичный разделитель. Сумма берётся по колонке `USD`. Всё считается локально в браузере — данные никуда не уходят.

## Проверено

На карте из видео: траты auth = 207.64$ (совпадает со сводкой админки), баланс ≈ 0.
